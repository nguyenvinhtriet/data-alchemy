# Python cho Khoa học Dữ liệu

Python đã trở thành ngôn ngữ thống trị trong lĩnh vực khoa học dữ liệu nhờ vào hệ sinh thái thư viện phong phú.

## Các thư viện quan trọng
- **Pandas**: Xử lý dữ liệu bảng.
- **NumPy**: Tính toán số học.
- **Matplotlib**: Vẽ biểu đồ.

# Python for Data Science

Python has become the dominant language in the field of data science thanks to its rich ecosystem of libraries.

## Important Libraries
- **Pandas**: Tabular data processing.
- **NumPy**: Numerical computing.
- **Matplotlib**: Plotting and visualization.
