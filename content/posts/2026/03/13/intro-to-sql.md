# Giới thiệu về SQL

SQL (Structured Query Language) là ngôn ngữ không thể thiếu đối với bất kỳ ai làm việc với dữ liệu.

## Các câu lệnh cơ bản
- `SELECT`: Lấy dữ liệu
- `INSERT`: Thêm dữ liệu
- `UPDATE`: Cập nhật dữ liệu
- `DELETE`: Xóa dữ liệu

# Introduction to SQL

SQL (Structured Query Language) is an essential language for anyone working with data.

## Basic Commands
- `SELECT`: Retrieve data
- `INSERT`: Add data
- `UPDATE`: Update data
- `DELETE`: Delete data
